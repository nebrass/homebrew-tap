# nebrass/homebrew-tap

Homebrew tap for `zornmesh` and related local-first agent mesh tools.

## Install

```bash
brew install nebrass/tap/zornmesh
```

Formulae are auto-bumped by the [zornmesh release workflow](https://github.com/nebrass/zorn-mesh/blob/main/.github/workflows/release.yml) on every `v*.*.*` tag.
